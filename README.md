# bank-reconciliation-tool
Automated Bank Reconciliation Tool with matching, classification, and Excel reporting (BlackLine-style)

This project is an automated **Bank Reconciliation Tool** built using Python and Streamlit.  
It is designed to replicate a simplified version of enterprise tools like BlackLine by automating transaction matching, classification, and reconciliation reporting.

---

## 🚀 Features

- ✅ Upload Bank Statement & GL / Accounting data
- 🔄 Automatic transaction matching (Amount + Date logic)
- 📊 Reconciliation Summary (Books vs Bank)
- 🏷 Classification of unmatched items:
  - Bank Charges
  - Interest
  - Outstanding Cheques
- 🧾 Automated Comments for unmatched transactions
- 📄 Excel Output with:
  - Reconciliation Summary
  - Matched Transactions
  - Unmatched Bank Entries
  - Unmatched GL Entries
  - Comments Sheet
- 🏢 Auto-detection of Company Name

---

## 🛠 Tech Stack

- Python
- Pandas
- Streamlit
- OpenPyXL / XlsxWriter
